# visualization

可视化项目主要功能是让用户能够创建自己的可视化页面,在页面中用户可以创建各种echarts图表,各种动画效果,还可以引入页面等等,此外还可以
echarts图表跟自己的数据库的数据相关联,将数据通过图表表现出来.还有报表工具将数据库的数据通过报表展示出来.
